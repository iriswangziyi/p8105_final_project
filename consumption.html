Consumption

This graph shows data from the Shipments of Malt Beverages and Per Capita Consumption by State document. This consumption is by captiate and only considers individuals who are 21 years old or older.

state_df %>%
  filter(year > 2009) %>%
  mutate(text_label = str_c("state: ", state)) %>% # creating text label 
  plot_ly( #using plotly to plot he tmax and tmin 
    x = ~year, y = ~consumption, color = ~factor(state), text = ~text_label,
    alpha = .5, type = "scatter", mode = "lines", colors = "viridis") %>%
  layout( # adding a title 
    title = "Consumption")
